# ⚽ Football Ticket Booking System – SQL Project

## 🚀 Overview
A SQL-based project that simulates a real-world football ticket booking system. It manages users, matches, and ticket bookings using a relational database structure.

---

## 📁 Project Files
- 🧱 `creating_table.sql` → Database schema (tables, keys, constraints)
- 📥 `value_insertion.sql` → Sample data for testing
- 🔍 `Queries.sql` → Data retrieval & analysis queries

---

## ✨ Key Features
- 👤 User management system
- ⚽ Match scheduling & categories
- 🎟️ Ticket booking system
- 💳 Payment status tracking
- 📊 Basic reporting & insights

---

## 🛠️ SQL Concepts Used
- 🔗 INNER JOIN & LEFT JOIN
- 🔍 WHERE filtering & LIKE search
- 📊 Aggregate functions (AVG)
- 🧠 Subqueries
- ⚠️ NULL handling with COALESCE
- 📑 Sorting & pagination (ORDER BY, LIMIT, OFFSET)

---

## 🎯 Purpose
To practice real-world SQL database design and querying by simulating a football ticket booking platform.
